📌 Project Description

This project explores the transition from traditional database storage to a blockchain-based system for recording transaction data. Instead of storing records in a centralized database, each transaction is securely written into a block, ensuring data integrity, transparency, and immutability.

Each block represents a transaction and contains detailed information such as transaction ID, sender, receiver, timestamp, and a breakdown of purchased items. The structure ensures that all transaction data is verifiable and tamper-resistant.

🔗 Example Block Structure
Tx ID: tx_001
Sender: Hengky
Receiver: CafeA
Recorded Amount: 45000
Timestamp: 2026-04-17T10:14:47.495Z
Computed Total: 45000

Items:
- Tea   : 20000 x 1
- Bread : 25000 x 1

🚀 Key Features
Replacement of centralized database with blockchain storage
Immutable transaction records
Transparent and traceable data flow
Automatic total computation validation
Itemized transaction details within each block
🎯 Objective

The goal of this project is to demonstrate how blockchain technology can be used as a secure alternative to traditional databases, especially for transactional systems where trust, integrity, and auditability are critical.
